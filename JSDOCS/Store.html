new Store(name, callback)

Source:

Crée un nouvel objet de stockage côté client et crée un jeu de données vide si aucun n'existe.
Parameters:
Name Type Description
name string le nom de la Base de Données que nous souhaitons utiliser
callback function Notre BD utilise les callbacks mais en condition réelle nous ferions probablement des appels AJAX

(static) find(query, callback)

Source:
  • store.js, line 29
Trouve les items en fonction d'une requête faite sous la forme d'un objet JS.
Example
db.find({foo: 'bar', hello: 'world'}, function (data) {
	 // data retournera tous les éléments qui ont foo: bar et
	 hello: world dans leurs propriétés
});
Parameters:
Name Type Description
query object La requête à comparer ({foo: 'bar'})
callback function callback appelée quand la requête est terminée
