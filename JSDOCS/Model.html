(static) read(queryopt, callbackopt)

Source:
  • model.js, line 33
Trouve et renvoie un modèle dans le storage. Si aucune requête n'est précisée, il va simplement tout retourner. Si nous passons une chaîne ou un numéro, cela sera considéré comme étant l'identifiant du modèle à trouver. Enfin, nou spouvons lui passer un objet.
Example
model.read(1, func); // Trouvera le modèle avec un ID de 1
model.read('1'); // comme ci-dessus
//Nous trouvererons ci-dessous un modèle avec foo égale à bar et un hello égale à world.
model.read({ foo: 'bar', hello: 'world' });
Parameters:
Name Type Attributes Description
query string | number | object <optional>
 Requête servant de référence pour la recherche
callback function <optional>
 Callback à appeler après que la tâche a été trouvée
